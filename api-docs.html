Example Usage

cURL

curl -X POST https://your-server.com/api/tool/json \
  -H "Content-Type: application/json" \
  -d '{"input": "{\"name\":\"test\"}"}'

JavaScript

const res = await fetch('https://your-server.com/api/tool/json', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({input: '{"name":"test"}'})
});
const data = await res.json();
console.log(data.result);

Python

import requests
res = requests.post('https://your-server.com/api/tool/json', 
  json={'input': '{"name":"test"}'})
print(res.json()['result'])
