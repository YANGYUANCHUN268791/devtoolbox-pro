# Coding Prompts (120+)

## Code Generation

### API Generator
Prompt: You are a senior backend engineer. Create a RESTful API using [language/framework] with:
- JWT authentication (access + refresh tokens)
- CRUD operations for [resource]
- Search with pagination
- Error handling middleware
- Input validation
- OpenAPI/Swagger documentation

### Database Schema Designer
Prompt: You are a database architect. Design schema for [project]:
- ER diagram (Mermaid format)
- SQL CREATE statements
- Index strategy
- Migration scripts

### CI/CD Pipeline
Prompt: Create a CI/CD pipeline for [platform]:
- Linting stage
- Unit + integration tests  
- Build and package
- Deploy to [environment]
- Health check
- Rollback strategy

## Debugging & Troubleshooting

### Performance Analysis
Prompt: Analyze performance issue: [description]
Investigate: CPU hotspots, memory leaks, I/O bottlenecks, slow queries, network latency

### Security Audit
Prompt: Audit code for: SQL injection, XSS, CSRF, auth bypass, sensitive data exposure, SSRF
Rank by CVSS score, provide fixed code

## Refactoring

### Legacy System
Prompt: Design refactoring plan for legacy codebase
Deliverable: phased roadmap, test strategy, risk assessment, effort estimation

## Documentation

### API Documentation
Prompt: Generate OpenAPI 3.0 spec for [endpoint]
Include: params, schemas, auth, error codes, examples

### Technical Design Doc
Prompt: Write technical design document for [project]
Format: background, architecture, core flow, DB design, API design, security, deployment
