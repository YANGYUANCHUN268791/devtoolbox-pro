# Design & Creative Prompts (60+)

## Product Design Review
Prompt: Review this design: [description]
Evaluate: information architecture, interaction flow, visual hierarchy, consistency, accessibility
Score 1-10 with suggestions

## Design System  
Prompt: Build design system for [product]:
- Design tokens (colors, typography, spacing)
- Component specs
- Typography scale
- Color system
- Icon style guide

## Brand Identity
Prompt: Design brand identity for [company]:
- Logo concepts (3 with rationale)
- Color palette
- Typography selection
- Brand voice guidelines
- Application examples

## UI Component Design
Prompt: Design [component] for [product type]
Deliver: wireframe, visual design, interaction states, responsive variants, accessibility notes
