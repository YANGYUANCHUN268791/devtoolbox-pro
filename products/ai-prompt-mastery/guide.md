# User Guide

## Quick Start
1. Open any prompts category file
2. Copy a prompt template
3. Replace [placeholders] with your specific content
4. Paste into ChatGPT, Claude, Gemini, or any AI tool

## Pro Tips
- Be specific with role definitions for higher quality
- Specify output format for immediately usable results
- Break complex tasks into multiple chained prompts
- Combine prompts across categories for best results
